# AI Smart Reply — public pages

Static **Privacy Policy** and **Support** pages for the AI Smart Reply Outlook add-in (R-Technologies), published with **GitHub Pages**.

## GitHub Pages setup

**Settings → Pages → Build and deployment**

- **Source:** Deploy from a branch  
- **Branch:** `main`  
- **Folder:** **`/ (root)`** — HTML files live at the repository root (`privacy.html`, `support.html`, `index.html`).

If you use **`/docs`** instead, GitHub only publishes files inside the `docs/` folder; root URLs like `/privacy.html` will 404 unless those files exist at the site root for your chosen source.
